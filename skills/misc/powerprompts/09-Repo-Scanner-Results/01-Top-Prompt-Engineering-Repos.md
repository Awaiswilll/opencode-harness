| **URL** | https://github.com/stanfordnlp/dspy |
| **Topics** | `prompt-optimization`, `llm`, `programming`, `stanford` |
| **Last Updated** | Very Active |

**Description:** Framework for algorithmically optimizing LM prompts and weights — widely called "PyTorch for prompts." Declarative programming paradigm for LMs.

**Key Features:**
- Prompt optimizers (BootstrapFewShot, MIPROv2, Bayesian)
- Signature-based module declaration
- Compilation pipeline for prompt tuning
- Multi-step pipeline orchestration
- Evaluation-driven optimization loop
- Extensive model and API support

**Why It's Useful:** The most important prompt engineering tool in existence. Changes the paradigm from manual prompt writing to programmatic optimization. If you use one advanced tool, use this.
